文法詳細

ないと
naito
harus melakukan; kecuali jika-
Halaman 139

📝 Cara Penggunaan

Kata Kerja (bentuk ない) + と

Kata Benda + でない / じゃない + と

な-adjective + でない / じゃない + と

い-adjective + くない + と

ないと (naito). Makna 1: harus melakukan (singkatan dari ないといけない). Makna 2: kecuali jika; kalau tidak... (akibat buruk).

📖 Contoh Kalimat

1もう行かないと...
もう いかないと...
Mou ikanai to...
Aku harus pergi sekarang...
2宿題をやらないと。
しゅくだいを やらないと。
Shukudai o yaranai to.
Aku harus mengerjakan PR.
3もう暗くなって、帰らないと。
もう くらくなって、かえらないと。
Mou kuraku natte, kaeranai to.
Sudah gelap, aku harus pulang.
4早くしないと間に合わないよ。
はやく しないと まにあわないよ。
Hayaku shinai to maniawanai yo.
Kalau tidak cepat, kamu tidak akan tepat waktu.
5早く寝ないと、明日起きられないよ。
はやく ねないと、あした おきられないよ。
Hayaku nenai to, ashita okirarenai yo.
Kalau tidak tidur cepat, besok tidak bisa bangun.
6この値段より安くないと買わない。
この ねだんより やすくないと かわない。
Kono nedan yori yasuku nai to kawanai.
Aku tidak akan beli kecuali lebih murah dari harga ini.
7JLPT先生でないと日本語を勉強しません。
JLPTせんせいでないと にほんごを べんきょうしません。
JLPT sensei denai to nihongo o benkyou shimasen.
Aku tidak akan belajar bahasa Jepang kecuali gurunya JLPT Sensei.
8この機会を取らないと他人に取られますよ。
この きかいを とらないと たにんに とられますよ。
Kono kikai o toranai to tanin ni toraremasu yo.
Kalau kamu tidak ambil kesempatan ini, akan diambil orang lain.
